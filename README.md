# 🔐 Login & Sign Up Interface

Uma interface de autenticação moderna e responsiva, com transições suaves entre as telas de login e cadastro. Este projeto foi desenvolvido com foco em UI/UX, utilizando animações para alternar os formulários de maneira elegante.

---

## 🚀 Tecnologias

Este projeto foi construído com as seguintes tecnologias:

*   **React** (com Vite)
*   **TypeScript**
*   **Styled Components** (CSS-in-JS)
*   **Ionicons**

## 🛠️ Funcionalidades

*   **Alternância de Telas:** Transição animada entre os formulários de Login e Cadastro.
*   **Design Moderno:** Interface limpa com paleta de cores em tons de roxo e cinza.
*   **Responsividade:** Layout adaptável para diferentes tamanhos de tela.
*   **Lógica de Transição:** Uso de propriedades dinâmicas no Styled Components para controlar as animações de `transform`.

## 📁 Estrutura de Arquivos Principal

*   `src/App.tsx`: Contém toda a lógica de estado (`toggled`) e os componentes estilizados.
*   `src/main.tsx`: Ponto de entrada da aplicação React.
*   `vite.config.ts`: Configurações de build do Vite.

## 🔧 Como Executar o Projeto

1.  **Clone o repositório:**
    ```bash
    git clone [https://github.com/Mary9998/Pagina-Login.git](https://github.com/Mary9998/Pagina-Login.git)
    ```

2.  **Entre na pasta do projeto:**
    ```bash
    cd Pagina-Login
    ```

3.  **Instale as dependências:**
    ```bash
    npm install
    ```

4.  **Inicie o servidor de desenvolvimento:**
    ```bash
    npm run dev
    ```

5.  **Acesse no seu navegador:**
    `http://localhost:5173`
